The aim is to analyze customer reviews and to find insights into product
ratings.

\section*{Processing \& Analysis}

The sample dataset used in the analysis contains 65 entries with 6
fields. It contains product reviews from 4 categories. Here following
operations were done on the data:

\begin{itemize}
\tightlist
\item
  Rows with missing values in any of the following columns were removed:

  \begin{itemize}
  \tightlist
  \item
    product\_id
  \item
    product\_name
  \item
    category
  \item
    customer\_id
  \item
    score
  \end{itemize}
\item
  Duplicate reviews were removed if the review was posted by same person
  in a given product
\end{itemize}

Then box plot showing distribution of review scores in each category and
a barchart showing total number of reviews in each category was plotted.
